    4. 배포: `flyctl deploy --remote-only`
       - `--remote-only`: 로컬 Docker 불필요, Fly.io 빌더 사용
    5. 배포 상태 확인: `flyctl status`
       - Machine이 running 상태인지 확인

    주의: Fly.io free tier는 카드 등록 필요할 수 있음. 실패 시 사용자에게 안내.
