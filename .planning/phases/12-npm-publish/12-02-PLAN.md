  <action>
1. 빌드 실행: `npm -w packages/client run build`

2. publish dry-run 실행하여 최종 확인:
   ```
   cd packages/client && npm publish --dry-run --access public
   ```
   - 포함 파일: bin/hivechat.js, dist/index.mjs (or .js), package.json
   - 크기: 1MB 미만
   - 이름: hivechat
   - 버전: 1.2.0

3. npm whoami 실행하여 인증 상태 확인:
   - 인증 안 되어 있으면 사용자에게 `npm login` 실행 요청 (checkpoint로 전환)

4. 인증 확인 후 실제 publish:
   ```
   cd packages/client && npm publish --access public
   ```

NOTE: "hivechat" 이름이 npm에서 사용 가능한지 publish시 확인됨. 이름 충돌 시 사용자와 상의 필요.
