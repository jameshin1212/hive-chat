**입력 최소 높이 보장 (RESP-02):**

ChatScreen.tsx의 messageAreaHeight 계산 (현재 line 84) 수정:

현재: `const messageAreaHeight = Math.max(1, rows - 4 - overlayHeight);`

변경:
```typescript
const MIN_INPUT_AREA = 3; // 최소: separator(1) + statusbar(1) + separator(1) + input(1) = 4, 하지만 이미 -4로 빠져있음
// messageArea가 너무 커서 input이 밀리는 것을 방지
// rows 전체에서 고정 영역(statusbar 2줄 separator + statusbar + input = 4줄)과 overlay를 빼되,
// messageArea가 최소 1줄, 그리고 overlay가 가용 공간을 초과하면 overlay를 줄임
const availableForMessages = rows - 4 - overlayHeight;
const messageAreaHeight = Math.max(1, Math.min(availableForMessages, rows - 5));
```

핵심: overlayHeight가 rows-4보다 클 경우에도 messageAreaHeight가 최소 1을 보장.
현재 Math.max(1, ...) 로 이미 최소 1줄 보장되어 있으나, overlayHeight 자체도 가용 공간에 맞게 cap:

```typescript
const maxOverlayHeight = Math.max(0, rows - 5); // 최소 messageArea 1줄 + input 보장
const effectiveOverlayHeight = Math.min(overlayHeight, maxOverlayHeight);
const messageAreaHeight = Math.max(1, rows - 4 - effectiveOverlayHeight);
```

이렇게 하면 터미널이 아무리 작아도 메시지 영역 1줄 + 입력 영역이 보장됨.
