  <action>
    1. Create `packages/server/src/GeoLocationService.ts`:
       - `normalizeIp(ip: string): string` -- strip "::ffff:" prefix
       - `lookupIp(ip: string): GeoResult | null` -- use geoip-lite. For private/loopback IPs: check DEV_GEO_LAT/DEV_GEO_LON env vars for dev fallback. If no env vars and private IP, return null.
       - `getDistanceKm(a, b): number` -- haversine with {unit: 'km'}
       - `findNearbyUsers(origin, users: Map, radiusKm): Array<{id, distance}>` -- iterate users, calc distance, filter by radius, sort ascending by distance. Round distance to 1 decimal.
       - IPv6 city lookup: per research, geoip-lite only supports IPv4 city lookup. Pure IPv6 returns null, handled by fallback.

    2. Create `packages/server/src/PresenceManager.ts`:
       - Class with `private users = new Map<string, UserRecord>()`
       - `register(id, record)`: If existing ID, call old ws.terminate() first, then set new. Return void.
       - `unregister(id)`: Delete from map
       - `getUser(id)`: Return user or undefined
       - `heartbeat(id)`: Update lastHeartbeat to Date.now()
       - `checkStaleUsers(): string[]`: Iterate users. If `Date.now() - lastHeartbeat > STALE_TIMEOUT_MS` and status is 'online', set status to 'offline', collect ID. Return stale IDs array.
       - `getNearbyUsers(originId, radiusKm): NearbyUser[]`: Get origin user coords, build location map of other online users, call findNearbyUsers, map results to NearbyUser shape with nickname/tag/aiCli/distance/status.
       - `getAllOnlineUsers(): UserRecord[]`: Filter by status 'online'
       - `getUserCount(): number`
       - Import STALE_TIMEOUT_MS from @cling-talk/shared constants
