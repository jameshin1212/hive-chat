    - Emits 'connected' event after receiving REGISTERED response
    - Emits 'reconnecting' event when connection drops
    - Emits 'disconnected' event after max reconnect failures or explicit disconnect()
    - Emits 'nearby_users' event with NearbyUser[] on NEARBY_USERS message
    - Emits 'user_joined' event with NearbyUser on USER_JOINED message
    - Emits 'user_left' event with {nickname, tag} on USER_LEFT message
    - Emits 'user_status' event with {nickname, tag, status} on USER_STATUS message
    - Emits 'error' event with {code, message} on ERROR message
    - Sends heartbeat every 30s while connected
    - requestNearbyUsers(radiusKm) sends GET_NEARBY message
    - updateRadius(radiusKm) sends UPDATE_RADIUS message
    - Reconnect uses exponential backoff: 500ms * 2^attempt, capped at 30s, with 0.5-1.5x jitter
    - disconnect() cleanly closes WebSocket without reconnection
    - getReconnectDelay() is deterministic given attempt number (minus jitter)
  </behavior>
  <action>
    1. Create `packages/client/src/network/SignalingClient.ts`:
       - Extends EventEmitter (from 'events')
       - Constructor: `(serverUrl: string, identity: Identity)`
       - Private fields: ws, reconnectAttempt, reconnectTimer, heartbeatTimer, isIntentionalClose
       - `connect()`:
         a. Create new WebSocket(serverUrl) -- use 'ws' package (isomorphic)
         b. On 'open': send REGISTER message with {type: MessageType.REGISTER, nickname: identity.nickname, tag: identity.tag, aiCli: identity.aiCli, protocolVersion: PROTOCOL_VERSION}
         c. On 'message': JSON.parse, validate with serverMessageSchema.safeParse(). Route by type:
           - REGISTERED: reset reconnectAttempt to 0, emit 'connected' + 'nearby_users' with response.users
           - NEARBY_USERS: emit 'nearby_users' with msg.users
           - USER_JOINED: emit 'user_joined' with msg.user
           - USER_LEFT: emit 'user_left' with {nickname: msg.nickname, tag: msg.tag}
           - USER_STATUS: emit 'user_status' with {nickname: msg.nickname, tag: msg.tag, status: msg.status}
           - ERROR: emit 'error' with {code: msg.code, message: msg.message}
         d. On 'close': if not intentional close, emit 'reconnecting', schedule reconnect
         e. On 'error': log, let close handler manage reconnection

       - `startHeartbeat()`: setInterval every HEARTBEAT_INTERVAL_MS, send {type: MessageType.HEARTBEAT}
       - `stopHeartbeat()`: clearInterval

       - `getReconnectDelay(): number`:
         base = RECONNECT_BASE_DELAY_MS * 2^reconnectAttempt
         capped = Math.min(base, RECONNECT_MAX_DELAY_MS)
         jitter = 0.5 + Math.random()
         return Math.floor(capped * jitter)

       - `scheduleReconnect()`:
         reconnectAttempt++
         delay = getReconnectDelay()
         reconnectTimer = setTimeout(() => connect(), delay)

       - `requestNearbyUsers(radiusKm: number)`: send {type: MessageType.GET_NEARBY, radiusKm}
       - `updateRadius(radiusKm: number)`: send {type: MessageType.UPDATE_RADIUS, radiusKm}

       - `disconnect()`:
         isIntentionalClose = true
         stopHeartbeat()
         clearTimeout(reconnectTimer)
         ws?.close()

       - `send(msg: ClientMessage)`: if ws.readyState === OPEN, ws.send(JSON.stringify(msg))
